102 Etiquetas HTML

Nombre de la Etiqueta Descripción Ejemplo de Uso
1 a Define un hipervínculo. Se utiliza para enlazar una página a otra o un recurso. <a href="https://www.example.com">Visita Ejemplo</a>

7 acronym La etiqueta de acrónimo en HTML se utiliza para definir el acrónimo que proporciona información útil a los navegadores, sistemas de traducción y motores de búsqueda. <acronym title=""> … </acronym>
8 address La etiqueta de dirección en HTML indica la información de contacto de una persona o una organización. <address> … </address>
9 title La etiqueta de title en HTML se utiliza para definir el título del documento HTML. Establece el título en la barra de herramientas del navegador. <title> Title name </title>
10 applet La etiqueta applet en HTML se utilizó para incrustar applets de Java en cualquier documento HTML y se dejó de usar a partir de HTML 5. <applet>...</applet>
11 area Esta etiqueta de área se utiliza en un documento HTML para mapear una parte de una imagen para que el usuario final pueda hacer clic en ella. <area>
12 article La etiqueta <article> es uno de los nuevos elementos de segmentación de HTML5. La etiqueta se utiliza para representar un artículo. <article>...</article>
13 aside La etiqueta <aside> se utiliza para describir el objeto principal de la página web de forma más corta, como un resaltador. <aside>...</aside>
14 audio Es una etiqueta útil si desea agregar audio como canciones, entrevistas, etc. en su página web. <audio>...</audio>

16 basefont Esta etiqueta se utiliza para establecer el color de texto, el tamaño de fuente y la familia de fuentes predeterminados de todo el texto del navegador. No es compatible con HTML5. <basefont>
17 bdi La etiqueta bdi hace referencia al aislamiento bidireccional. Diferencia un texto de otro texto que puede estar formateado en una dirección diferente. ...

21 blockquote La etiqueta blockquote en HTML se utiliza para mostrar las citas largas (una sección que se cita de otra fuente). <blockquote>Contenido...</blockquote>
22 body La etiqueta del cuerpo en HTML se utiliza para definir el contenido principal presente dentro de una página HTML. <body>Contenido...</body>
23 b La etiqueta negrita en HTML se utiliza para especificar el texto en negrita sin ninguna importancia adicional. <b>...</b>
24 br La etiqueta de salto inserta un único salto de línea o retorno de carro en el documento. Este elemento no tiene etiqueta de cierre. <br>
25 button La etiqueta de botón en HTML se utiliza para definir el botón en el que se puede hacer clic. La etiqueta <button> se utiliza para enviar el contenido. <button type="button">...</button>

42 div La etiqueta div se utiliza en HTML para realizar divisiones de contenido en la página web (texto, imágenes, encabezado, pie de página, barra de navegación, etc.). <div>Contenido</div>
